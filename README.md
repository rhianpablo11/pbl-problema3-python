# pbl-problema3-python
Projeto 3 do PBL de Algoritmos e Programação 01
# Para executar o projeto
1; Clone o projeto

```sh
git clone https://github.com/rhianpablo11/pbl-problema3-python.git
```

2; Abra a pasta do projeto

```sh
cd pbl-problema3-python
```

3; Execute o comando

```sh
python sistema_de_pesquisa.py help
```
